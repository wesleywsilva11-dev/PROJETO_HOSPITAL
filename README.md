# Sistema de Atendimento Hospitalar
## Sobre o projeto
Este projeto foi desenvolvido em Python com o objetivo de simular um sistema de atendimento hospitalar. Nele é possível cadastrar pacientes, organizar a fila de atendimento de acordo com a prioridade e visualizar os pacientes cadastrados.
